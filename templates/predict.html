CIFAR-10 Classifiation

This is a classification software for CIFAR-10 dataset. You can upload images and model will be able to predict name of the object in the image.

About the Model

I have used a simple CNN model for training and testing on the dataset. The model achieved an accuracy of 80% on both train and test data.

The model is able to predict 10 types of objetcs namely:

  • Airplane
  • Automobile
  • Bird
  • Cat
  • Deer
  • Dog
  • Frog
  • Horse
  • Ship
  • Truck
